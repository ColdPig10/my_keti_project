- `mcp_mirrorlake_server_sse.py` : SSE 기반 MCP 서버 코드
- `gradio_langgraph_mcp_sse_client.py` : LangGraph-챗봇 Gradio 클라이언트
- `self_rag.py`, `parkinglot_law.pdf` : RAG 도구 및 테스트 데이터
